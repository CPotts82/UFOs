# UFOs
## Overview
The purpose of this project was to create a user-friendly, interactive web application to filter and display recent UFO sighting information from a very large dataset. Using JavaScript and Bootstrap, an aesthitically pleasing and easy to use web application make filtering for unique UFO data interesting and entertaining.
